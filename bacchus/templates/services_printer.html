프린터 사용 방법

계정 생성

프린터를 이용하기 위해서는 신도리코 프린터 솔루션 계정이 필요합니다.

  • 기존 Netprint를 이용하던 고학번은 기존 Netprint 계정의 아이디로 비밀번호가 초기화 되어 있습니다.
  • 신규로 프린터를 이용하고 싶은 경우 SINDOH(링크) 에서 신규가입을 하여야 합니다.

카드 등록

프린터 이용시 계정 로그인도 가능하지만 카드 등록시 더욱 편리하게 이용 가능합니다. 카드 등록은 최초 1회만 하면 됩니다.

  • 프린터의 카드 태그부분에 학생증(또는 등록을 원하는 RFID카드)을 태그합니다.
  • 프린터 화면에서 로그인 합니다.
  • 정상적으로 등록이 되었다면 로그아웃을 해줍시다.

프린터 쿼터비 충전

쿼터비는 농협 302-0546-4286-71 로 입금 후, 입금 내역을 메일이나(esjee@cse.snu.ac.kr) 문자로(010-2287-2945)보내면 됩니다.

출력

  • 원하는 문서에서 인쇄를 선택하고 프린트 드라이버는 SINDOH uPrint를 선택합니다.
  • 로그인 팝업창이 뜨면 Public User 부분의 체크를 해제한 후 자신의 계정으로 로그인 합니다. 팝업창이 안보일 시 작업표시줄을 확인하시기 바랍니다.
  • 출력이 진행되면 프린터기로 가서 등록된 카드를 태그하거나 로그인을 해줍니다.
  • 로그인이 완료되면 청색 점등이 들어오며 출력이 진행됩니다.

복사

  • 등록된 카드를 태그하거나 로그인을 합니다.
  • 숫자키를 이용해 복사 매수를 설정합니다. 잘못 입력시 C(삭제)키를 눌러 재입력 합니다.
  • 시작 버튼을 누르면 복사가 진행됩니다.

개인 이용내역 확인

SINDOH(링크)에서 등록된 계정으로 로그인하면 이용내역을 확인하실 수 있습니다.

기타 문의사항

기타 프린터 클라이언트 문제가 발생할 시 바쿠스 홈페이지나 contact@bacchus.snucse.org로 연락해 주시기 바랍니다.
