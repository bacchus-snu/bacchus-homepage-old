사용 환경

shell 프로그램으로 bash와 zsh가 제공되지만, 로그인 shell은 bash로 고정되어 있습니다. 사용자의 필요에 맞게 .profile 등의 개인 설정 파일을 편집하여 사용하기 바랍니다.

자원 제한

Unix는 기본적으로 멀티유저를 위한 OS이므로, CPU나 메모리를 한 사람이 독점할 수 없습니다. 따라서 장시간 지나치게 자원을 점유하고 있는 프로세스는 다른 사용자를 위해 관리자가 임의로 제거합니다. 좀비프로세스이거나 CPU의 20% 이상을 1시간 이상 점유하고 있을 시는 임의로 제거 할 수 있음을 유념해 주시기 바랍니다.

프로그램 신청

  • Linux는 개인 사용자가 자신의 계정으로 프로그램을 설치할 수 있으므로 가급적 자신의 계정에 설치하는 것을 원칙으로 합니다.
  • 반드시 설치가 필요한 경우, 사용자가 프로그램 설치 신청을 하면 됩니다.

DB 계정 신청

  • 신청자에 한해 Mimosa에서 사용할 수 있는 MySQL DB와 계정을 발급해 드립니다.
  • 필요한 경우, 사용자가 DB 계정 신청을 하면 됩니다.

백업

1차적인 백업의 의무는 사용자에게 있습니다.

접속 제한(보안 관련)

정보의 보안과 암호화를 위해 telnet과 ftp는 이용하실 수 없습니다. 모든 작업은 Secure Shell과 sftp로 접속하여 진행하셔야 합니다.
